Projects

GUNNER Z

Using the java eclipse IDE, I created a first person shooter game called 'GUNNER Z'. I was inspired to create the game after learning about a technique called ray casting, which many of the old fps games used. The actual game was closely inspired by Call of Duty Zombies, and has the player face off against endless waves of enemies. The game also implements path finding AI to lead enemies towards the player. You can view the project on github here.
